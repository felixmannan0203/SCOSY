Security Audit Log

Active Security Measures

PBKDF2 Password Hashing Active

All passwords are hashed using PBKDF2 with SHA-256, 100,000 iterations, and unique 128-bit random salts.

AES-256-GCM Data Encryption Active

All local storage data is encrypted using AES-256-GCM with keys derived from your session.

Session Timeout & Rotation Active

Sessions automatically expire after 30 minutes of inactivity. Tokens are cryptographically random 256-bit values.

Rate Limiting & Lockout Active

Accounts are locked for 15 minutes after 5 failed login attempts.

CSRF Token Protection Active

All state-changing forms include cryptographically random CSRF tokens validated on submission.

XSS Prevention Active

All user inputs are sanitized using HTML entity encoding and CSP headers block inline scripts.

Anonymous Isolation Active

Anonymous complaints are stored in a separate encrypted namespace with no linkage to sessions.

Encrypted Backup Integrity Active

Backups use AES-256-GCM with authenticated encryption. Tampered files are detected and rejected.
